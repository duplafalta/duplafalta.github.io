Installation Instructions

  1. Copy (upload) the "menu" folder to the root directory of your website.

  2. Copy and paste the code [1] below where you want your menu to show up. This code can be placed anywhere between the <body> and </body> tags.



    [1] The code above consists of the two parts:

    1. The <script> tag adds a reference to the main JavaScript file "menu.js".
      If the page is located in a subfolder, use the src with two dots: "../menu/menu.js".

    2. The <div> element is a container for the menu. It includes the name of your menu ("menu") as an identifier.
